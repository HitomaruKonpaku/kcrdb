# kcrdb

KanColle Replay DB

## Environment

| Key           | Desc                  | Default                                             |
|---------------|-----------------------|-----------------------------------------------------|
| NO_COLOR      | Disable log color     |                                                     |
| NO_TIMESTAMP  | Disable log timestamp |                                                     |
| DATABASE_TYPE |                       | postgres                                            |
| DATABASE_URL  |                       | postgresql://postgres:admin@localhost:5432/postgres |
| REDIS_URL     |                       | redis://localhost:6379                              |
| CACHE_TTL     |                       | 1m                                                  |
